# AETHER

**Internet curiosities, found.**

A high-end, minimalistic gateway to a curated collection of the web's most interesting, interactive, and obscure corners.

## Live Site
[View AETHER](https://sfmqrb.github.io/web-explorer/)

## Features
- **The AETHER Gate**: A one-click "Launch" into a random verified destination.
- **Focus Arc**: A hidden, rotary dial explorer revealed only on hover.
- **Infinite Loop**: Seamlessly scroll through 100+ curated nodes forever.
- **Minimalist Aesthetic**: 8rem typography, glassmorphism, and a "Lazy Doc" approach.

## Lazy Doc
1. Click **Launch** for a random teleport.
2. Hover the **Right Edge** of the screen and scroll to browse the full dial (desktop).
3. Tap the **☰ button** (bottom-right) to open the explorer on mobile.
4. Click any node in the dial/list to jump directly.
